# Task 22 - Release Preparation

## Goal
Prepare the Android build for distribution.

## Steps
1. Configure signing keystore.
2. Update `AndroidManifest` with app name, icon, and permissions.
3. Generate release AAB via Gradle.
4. Verify installation on physical device.

## Output
- Play Store-ready bundle.

## Notes
Include privacy disclosure for on-device AI if publishing publicly.
